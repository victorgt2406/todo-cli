# Todo cli
A simple todo cli written in go

## Installation
### Ubuntu
```bash
curl https://victorgt2406.github.io/todo-cli/install.sh | bash
```
### MacOS
```bash
curl https://victorgt2406.github.io/todo-cli/install.sh | zsh
```

## Usage

### Create a task
```bash
tdc go to the market
```

### View tasks
```bash
tdc
```
