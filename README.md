# Todo cli
A simple todo cli written in go

## Installation

```bash
go install github.com/victorgt2406/todo-cli@latest
```

## Usage

### Create a task
```bash
tdc go to the market
```

### View tasks
List all relevant tasks
```bash
tdc l
```
List the important tasks
```bash
tdc li
```

### Delete tasks
Remove all tasks
```bash
tdc ra
```
Remove task by id
```bash
tdc r[TASK ID]
```

Remove task by slug
```bash
tdc r [task-name] # ℹ️ help your self using tab ->
```
